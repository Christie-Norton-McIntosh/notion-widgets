Notion Widgets

You can embed unlimited HTML widgets in Notion for free using GitHub Pages!
Since Notion doesn't allow direct HTML embedding, GitHub Pages provides a perfect workaround solution.

Create a GitHub Repository

  • Go to your GitHub profile
  • Click the '+' button in the top right corner
  • Select 'New Repository'
  • Give it a name and click 'Create Repository'

Create Your First HTML File

  • Click 'creating a new file'
  • Name it 'index.html'
  • Add your HTML code (start with something simple like "Hello World")
  • Click 'Commit new file'

Enable GitHub Pages

  • Go to repository Settings
  • Select 'Pages' from the left menu
  • Under 'Branch', select 'Main'
  • Click 'Save'
  • Wait for your site to be published (usually takes 1-2 minutes)

Embed in Notion

  • Copy your GitHub Pages URL
  • Go to Notion
  • Create an embed block
  • Paste the URL

Adding Custom Widgets

  • Create new HTML files in your repository
  • Add your widget's HTML code
  • Commit the changes
  • Use the URL pattern: your-github-pages-url/filename.html
